# Dashboard VacinaBR-PNI

Dashboard demonstrativo em Streamlit para explorar os CSVs em `data/analytics`.

## Uso

```powershell
pip install -r requirements-dashboard.txt
streamlit run dashboard/streamlit_app.py
```

O app le os CSVs analiticos ja gerados pela pipeline. No Colab ou em outro
ambiente, informe a pasta correta dos CSVs na barra lateral.
